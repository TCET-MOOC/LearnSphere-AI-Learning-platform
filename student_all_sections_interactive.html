L{f(t)} = ∫₀^∞ e^(-st) f(t) dt. Key pairs: L{1} = 1/s, L{e^(at)} = 1/(s−a), L{sin(at)} = a/(s²+a²)

Binary Search Tree — insertion logic
DSA · Lec 14 · ⏱ 08:45 · Yesterday
Timestamped⋯
If value < root → go left. If value > root → go right. Repeat until null node found. Insert there. Time complexity: O(log n) average, O(n) worst case (skewed tree).

First law of thermodynamics
Thermodynamics · Lec 3 · ⏱ 31:00 · 2 days ago
Timestamped⋯
ΔU = Q − W. Energy cannot be created or destroyed. Q = heat added to system, W = work done BY system. Sign conventions are critical!
